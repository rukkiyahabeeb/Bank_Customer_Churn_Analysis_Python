# Bank_Customer_Churn_Analysis_Python
An end-to-end data analysis project using Python to identify why customers are leaving the bank and providing data-driven strategies to improve retention.

## Project Overview:
  This project analyzes the bank-customer-churn dataset to uncover patterns behind customer attrition.By using Pandas for data manipulation and Matplotlib/Seaborn/Plotly for visualization,we identify high-risk segments and offer actionable business solutions.

## Key Visualizations & Insights:
 * Overall Churn: 20% of the customer base has exited.
 * Geography: Germany shows the highest churn rate despite having a similar salary distribution to other regions.
 * Demographics: Customers aged 40–55 and Female customers are significantly more likely to leave.
 * Product Usage: Customers with 2 products are the most loyal; those with 3 or more have a near 100% churn rate.
 * Financials: High-balance accounts ($100k+) are more volatile than zero-balance accounts.

## Dataset Structure:
The dataset includes the following features:
 * CustomerId,Gender,Age,Tenure,Balance,
 * Surname,NumOfProducts,HasCrCard,IsActiveMember,
 * CreditScore, EstimatedSalary,
 * Country,Churned(Target).

## Tech Stack:
 * Language: Python 3.x
 * Data Manipulation: Pandas, NumPy
 * Visualization: Matplotlib, Seaborn, Plotly (Interactive)
